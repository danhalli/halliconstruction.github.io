Carrie’s Garage-to-ADU Conversion Roadmap

Portland 2026 Code Compliant

Project Overview

This roadmap outlines the permitting, construction, and inspection process for converting an existing garage into a legal Accessory Dwelling Unit (ADU) in Portland, Oregon.

The project follows an Owner-Builder permit structure, where the homeowner acts as the permit holder for certain regulated trades, while Halli Construction completes all structural and finish work.

SDC Waiver

System Development Charges (SDCs) of $12,000–$20,000 are waived for ADUs when a covenant is recorded.

SDC Waiver Program Information

Critical Requirement:
The SDC Waiver Covenant must be signed, notarized, and recorded. Proof of recording is required before permits can be issued.

Mandatory Forms & Reference Documents

  • Residential Combination Building Permit Application
  • Owner Doing Work Statement
  • SDC Waiver & Covenant
  • Oregon Energy Code – Measure A
  • Site Plan / Erosion Control Guidance
